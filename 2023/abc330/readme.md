## C - Minimize Abs 2

正整数 D が与えられます。

非負整数 x,y に対する ∣x^2 +y^2 −D∣ の最小値を求めてください。

## 解説

全探索するとして、その範囲はどの程度か？

x^2 >= Dや y^2  >= D の時に終了する必要がなるので、

x >= √d + 1 や y >= √d + 1 を満たすときに終了する。

ただ、それだけだとTLEになるので、xのみを対象として全探索する。

d 以上で最も近いものと、d 以下で最も近いものを求める。

なので、移行して y^2 <= D - x^2の最大値を求める。（大体、√d-z^2 の切り捨て）

ついでに、それ + 1も試す。
